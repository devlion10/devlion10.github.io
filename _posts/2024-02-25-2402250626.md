# BASH  명령어로 GIT PULL , COMMIT, PUSH

**GIT PULL 및 PUSH**



- git init

```bash
git init
```

- 원격저장소 추가

```bash
git remote add origin <원격 저장소 URL>
ex> git remote add origin https://github.com/devlion10/map_front.git
```

- 원격저장소 보기

```bash
git remote -v
```

- 원격저장소 Pull

```bash
git pull origin master
```

- 원격저장소 commit

```bash
#전체 파일 stage에 add
git add *

#추가된 파일 스테이지 상태보기
git status

#커밋메시지
git commit -m "first commit"

#push
git push origin master
```
